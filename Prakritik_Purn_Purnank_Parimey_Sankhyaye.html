प्राकृतिक संख्याएँ

(Natural Numbers - N)

परिभाषा: गिनती करने के लिए जिन संख्याओं का प्रयोग किया जाता है, उन्हें प्राकृतिक संख्याएँ कहते हैं।

समुच्चय:

N = {1, 2, 3, 4, 5, ...}

मुख्य बिंदु:

सबसे छोटी प्राकृतिक संख्या 1 है।

इसमें 0 शामिल नहीं होता।

ऋणात्मक संख्याएँ शामिल नहीं होतीं।

उदाहरण: 7, 15, 100, 999

 पूर्ण संख्याएँ (Whole Numbers - W)

परिभाषा: प्राकृतिक संख्याओं के साथ 0 को मिलाने पर पूर्ण संख्याएँ बनती हैं।

समुच्चय:

W = {0, 1, 2, 3, 4, 5, ...}

मुख्य बिंदु:

सबसे छोटी पूर्ण संख्या 0 है।

सभी प्राकृतिक संख्याएँ पूर्ण संख्याएँ होती हैं।

ऋणात्मक संख्याएँ इसमें शामिल नहीं होतीं।

उदाहरण: 0, 8, 45, 200

पूर्णांक (Integers - Z)

परिभाषा: धनात्मक संख्याएँ, ऋणात्मक संख्याएँ तथा शून्य मिलकर पूर्णांक कहलाते हैं।

समुच्चय:

Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}

मुख्य बिंदु:

पूर्णांक धनात्मक, ऋणात्मक और शून्य तीनों होते हैं।

दशमलव एवं भिन्न पूर्णांक नहीं होते।

उदाहरण: -15, -2, 0, 9, 45

परिमेय संख्याएँ

(Rational Numbers - Q)

परिभाषा: जिन संख्याओं को p/q के रूप में लिखा जा सके, जहाँ q ≠ 0, उन्हें परिमेय संख्याएँ कहते हैं।

रूप:

Q = p/q, जहाँ q ≠ 0

मुख्य बिंदु:

सभी पूर्णांक परिमेय संख्याएँ हैं।

सभी भिन्न परिमेय संख्याएँ होती हैं।

समाप्त तथा आवर्ती दशमलव परिमेय होते हैं।

उदाहरण: 3/5, -7/4, 8 (=8/1), 0 (=0/1), 0.25

अपरिमेय संख्याएँ

(Irrational Numbers)

परिभाषा: जिन संख्याओं को p/q के रूप में नहीं लिखा जा सकता, वे अपरिमेय संख्याएँ कहलाती हैं।

मुख्य बिंदु:

दशमलव अनंत तथा अनावर्ती होते हैं।

इनका सटीक भिन्न रूप नहीं बनाया जा सकता।

उदाहरण: √2, √3, √5, π

परीक्षा में याद रखने योग्य तथ्य

सबसे छोटी प्राकृतिक संख्या = 1

सबसे छोटी पूर्ण संख्या = 0

सबसे बड़ा या सबसे छोटा पूर्णांक नहीं होता।

प्रत्येक प्राकृतिक संख्या पूर्ण संख्या है।

प्रत्येक पूर्ण संख्या पूर्णांक है।

प्रत्येक पूर्णांक परिमेय संख्या है।

प्रत्येक परिमेय संख्या वास्तविक संख्या है।

प्रत्येक वास्तविक संख्या परिमेय नहीं होती, क्योंकि अपरिमेय संख्याएँ भी वास्तविक होती हैं।
